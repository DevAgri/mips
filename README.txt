O projeto MIPS é uma solução composta por uma armadilha e sistemas para identificação e monitoramento de pragas em cultivos de soja.

Neste projeto temos os seguintes arquivos:

Coletor_Lazarus_ZeosLib.rar - é um prototipo inicial para editar os registros de banco com a identificação do que foi verificada em campo.
coletor_mips_20170611181354.zip - é um sistema para que seja feita a identificação visual da praga e envio de alertas - necessita de apache, mysql e php.
MIP_RC_20170611181811.zip - é um sistema para o acompanhamento do que foi coletado visualizando registros individuais de coleta ou através de gráficos  - necessita de apache, mysql e php.
mips.rar - é um sistema para a recepção de imagens de uma câmera localizada na armadilha e processamento da imagem, identificando a praga e quantificando a mesma, posteriormente envia os dados para o banco.
script_db.rar - é um pequeno script para a geração do banco de dados utilizados pelos sistemas acima.
codigo_arduino.ino - contém o código necessário para fazer a armadilha funcionar.
